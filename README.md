# NNotes_Scripts

## Instalacion
1. Clona este repositorio
```shell
git clone https://github.com/Delta27222/NNotes_Scripts.git && cd NNotes_Scripts
```

3. Cambia los permisos de ejecucion de install.sh
```shell
sudo chmod +x ./install.sh
```

3. Ejecuta install.sh
```shell
sudo chmod +x ./install.sh
```
